# プライバシーポリシー

本サービス「Get The Time」（以下「本サービス」といいます。）の運営者（以下「当方」といいます。）は、本サービスにおけるユーザーの個人情報を含む利用者情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

## 第1条（取得する情報）

当方は、本サービスの提供にあたり、以下の情報を取得します。

### (1) ユーザーが提供する情報

- メールアドレス
- ニックネーム（氏名・表示名）
- パスワード（暗号化して保存し、当方が平文を知ることはできません）
- Google アカウントによる認証を利用する場合、Google から提供される識別情報（プロフィール識別子・メールアドレス等）

### (2) ユーザーが本サービス上で記録する情報

- 光の時間（理想とする行動・なりたい自分に関する記録）
- 闇の時間（誘惑となる行動に関する記録）
- 活動記録（ポモドーロセッションの記録、タスク・コメント・各種評価値等）
- 後悔した1日の記録およびその要約
- 浄化タイマー・ポモドーロ設定等の利用状況に関する情報

これらの記録には、ユーザー自身の内面に関する機微な情報が含まれることがあります。当方は、これらを本サービスの提供目的の範囲内でのみ利用します。

### (3) 自動的に取得される情報

- アクセスログ、IP アドレス、ブラウザの種類、利用日時等
- Cookie および類似技術を用いて取得される情報（後述）

## 第2条（利用目的）

当方は、取得した情報を以下の目的で利用します。

1. 本サービスの提供・運営・本人認証のため
2. ユーザーからのお問い合わせに対応するため
3. 本サービスの機能（自己像形成度の計測、タイマー、各種記録の表示等）を提供するため
4. ユーザーが記録した「後悔した1日の記録」の要約を生成AIにより生成し、ユーザーの自己理解を支援するため
5. 本サービスの利用状況を分析し、機能改善・品質向上に役立てるため
6. 本サービスの障害対応・不具合調査・保守・維持のため
7. 利用規約に違反する行為への対応のため
8. 本サービスに関する重要なお知らせ等を通知するため

## 第3条（利用目的の変更）

1. 当方は、利用目的が変更前と関連性を有すると合理的に認められる場合に限り、個人情報の利用目的を変更するものとします。
2. 利用目的を変更した場合には、変更後の目的について、当方所定の方法により、ユーザーに通知し、または本サービス上に公表するものとします。

## 第4条（Cookie およびアクセス解析ツールの利用）

1. 本サービスは、サービスの利便性向上および利用状況の分析のため、Cookie を使用することがあります。
2. 本サービスは、アクセス解析ツールとして Google が提供する「Google Analytics」を利用しています。Google Analytics は Cookie を利用してユーザーの利用情報を収集しますが、これらの情報は匿名で収集されており、個人を特定するものではありません。
3. ユーザーは、ブラウザの設定により Cookie を無効化することで、これらの情報収集を拒否することができます。また、Google Analytics による収集は、Google が提供するオプトアウトアドオンを利用して停止することができます。
   - Google Analytics 利用規約：https://marketingplatform.google.com/about/analytics/terms/jp/
   - Google プライバシーポリシー：https://policies.google.com/privacy

## 第5条（第三者提供）

当方は、次に掲げる場合を除き、あらかじめユーザーの同意を得ることなく、個人情報を第三者に提供しません。

1. 法令に基づく場合
2. 人の生命・身体または財産の保護のために必要があり、本人の同意を得ることが困難な場合
3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要があり、本人の同意を得ることが困難な場合
4. 国の機関等が法令の定める事務を遂行することに協力する必要がある場合

## 第6条（外部サービスへの取り扱いの委託）

当方は、本サービスの運営に必要な範囲で、以下の外部サービスに個人情報の取り扱いを委託しています。これらの一部は日本国外に所在するサーバーで情報を取り扱う場合があります。

| 委託先サービス | 利用目的 | 備考 |
| --- | --- | --- |
| Render | 本サービスのホスティング | 米国 |
| Neon | データベースの提供 | 米国 |
| Resend | メールの送信 | 米国 |
| Google（Analytics / OAuth 認証） | アクセス解析・ログイン認証 | 米国 |
| OpenAI（OpenAI, L.L.C.） | 「後悔した1日の記録」の要約生成（生成AI） | 米国 |

ユーザーがお気に入り登録した「後悔した1日の記録」の本文は、要約の生成のために OpenAI の API に送信されます。当該データは、OpenAI のポリシーに基づいて取り扱われます。

当方は、委託先に対して個人情報を適切に取り扱うよう必要かつ適切な監督を行います。

## 第7条（個人情報の開示・訂正・削除）

1. ユーザーは、当方の保有する自己の個人情報について、開示・訂正・追加・削除・利用停止を請求することができます。
2. 請求を希望する場合は、本ポリシー末尾のお問い合わせ窓口までご連絡ください。本人確認のうえ、法令に従い合理的な期間内に対応します。
3. 前項の開示の請求があった場合でも、次のいずれかに該当する場合には、その全部または一部を開示しないことがあります。この場合、当方は遅滞なくその旨を通知します。
   1. 本人または第三者の生命、身体、財産その他の権利利益を害するおそれがある場合
   2. 当方の業務の適正な実施に著しい支障を及ぼすおそれがある場合
   3. その他法令に違反することとなる場合
4. ユーザーが本サービスを退会した場合、当方は、法令で保存が義務付けられる情報を除き、当該ユーザーの個人情報および投稿データを削除します。

## 第8条（個人情報の安全管理）

当方は、取得した個人情報の漏えい、滅失またはき損の防止その他の安全管理のために、必要かつ適切な措置を講じます。パスワードは暗号化して保存し、通信は SSL/TLS により暗号化します。

## 第9条（プライバシーポリシーの変更）

1. 本ポリシーの内容は、法令その他本ポリシーに別段の定めのある事項を除いて、ユーザーに通知することなく変更することができるものとします。
2. 当方が別途定める場合を除き、変更後のプライバシーポリシーは、本サービス上に掲載した時点から効力を生じるものとします。

## 第10条（お問い合わせ窓口）

本ポリシーに関するお問い合わせ、および個人情報の開示・訂正・削除・利用停止等の請求は、下記の窓口までお願いいたします。なお、当方の氏名および住所は、法令に基づき、本人からの請求に応じて遅滞なく開示いたします。

- 連絡先：getthetime.support@gmail.com
